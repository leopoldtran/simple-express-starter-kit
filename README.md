# Simple express server starter kit

## Dependencies
- dotenv
- express
- cors
- body-parser
- mongodb

## Dev dependencies
- babel-cli
- babel-core
- babel-eslint
- babel-preset-env
- eslint
- eslint-plugin-import
- nodemon

## Install
```
yarn
```

## Run script
```
yarn run-script
```

## Run with nodemon
```
yarn start
```
